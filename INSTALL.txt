 MDVSEC Secure Access Point - Quick Installation Guide
═══════════════════════════════════════════════════════════════════

📦 Package: mdvsec-ap_1.4.0_all.deb
🏗️  Architecture: Universal (x86, x64, ARM, ARM64)

═══════════════════════════════════════════════════════════════════
 INSTALLATION
═══════════════════════════════════════════════════════════════════

Option 1: Using apt (Recommended)
──────────────────────────────────
This automatically handles dependencies:

    sudo apt install ./dist/mdvsec-ap_1.4.0_all.deb

Option 2: Using dpkg
────────────────────
Install manually then resolve dependencies:

    sudo dpkg -i dist/mdvsec-ap_1.4.0_all.deb
    sudo apt-get install -f

═══════════════════════════════════════════════════════════════════
 RUNNING THE PROGRAM
═══════════════════════════════════════════════════════════════════

After installation:

    sudo mdvsec-ap

The program will display an interactive menu with options for:
  • Quick Start (automated setup)
  • WiFi Setup
  • Security Settings (DoH, VPN, IDS/IPS)
  • Admin Menu (monitoring, management)

═══════════════════════════════════════════════════════════════════
 BUILDING FROM SOURCE
═══════════════════════════════════════════════════════════════════

To build the package yourself:

1. Install build dependencies:
    sudo apt install dpkg-dev

2. Run the build script:
    ./build-deb.sh

3. Package will be created in dist/ directory

═══════════════════════════════════════════════════════════════════
 UNINSTALLATION
═══════════════════════════════════════════════════════════════════

Remove package (keep configuration):
    sudo apt remove mdvsec-ap

Remove package and all configuration:
    sudo apt purge mdvsec-ap

═══════════════════════════════════════════════════════════════════
 PACKAGE CONTENTS
═══════════════════════════════════════════════════════════════════

Installed files:
  • /usr/local/bin/mdvsec-ap (main executable)
  • /etc/secure-ap/ (configuration directory)
  • /usr/share/doc/mdvsec-ap/ (documentation)

Runtime files (created by script):
  • /usr/local/bin/mdvsec-notify
  • /usr/local/bin/mdvsec-monitor-clients
  • /usr/local/bin/mdvsec-wan-failover.sh
  • /etc/systemd/system/mdvsec-*.service
